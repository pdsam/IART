===================
== Para compilar ==
===================

Certificar-se que existe a pasta ./bin/
Se nao existir, criar um novo.

Correr: make release


==============
== Par usar ==
==============

Correr: ./bin/release

Aparecera uma interface para escolher o dataset.
Depois de escolher o dataset o programa pedir-lhe-a o algoritmo a usar.

Depois de escolher o algoritmo o programa pedira um conjunto de parametros
necessarios e depois correra o algoritmo para resover o problema.

No final e imprimida a solucao obtida, bem como algumas estatisticas.

Aparecera novamente o menu de escolha de algoritmo para poder escolher outro
algoritmo para correr nesse mesmo dataset.

A qualquer momento pode fazer CTRL+C para sair do programa.
