==============
== Par usar ==
==============

Correr: bin/release

Aparecera uma interface para escolher o dataset.
Depois escolher o dataset o programa pedir-lhe-a o algoritmo a usar.

Depois de escolher o algoritmo o programa pedira um conjunto de parametros
necessarios e depois correra o algoritmo para resover o problema.
